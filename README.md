# cubeTherm

A personal project for modelling the thermal behavior of a CubeSat in LEO.

**Status:** In early development 🚧

## Goal
To understand how a simple thermal model behaves and to develop it over time. The purpose of this file is not to produce flight-quality analyses but rather to be a learning experience for me in a field that I am interested in.
